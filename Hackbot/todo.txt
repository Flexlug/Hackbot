1. ЧЕЛОВЕК ПИШЕТ /START

	= Заригистрироваться на хакатон
	= Авторизоваться как участник хакатона

+ Если человек уже зарегистристрирован как участник хакатона (т.е. прошёл пункт 2), то его перекидывает сразу на пункт 3
+ Если человек НЕ зарегистристрирован, то он прозодит процесс регистрации в пункте 2.

1.1 ЕСЛИ ЖМЁТ "Заригистрироваться на хакатон", то ПУНКТ 2
1.2 ЕСЛИ ЖМЁТ "Авторизоваться как участник хакатона", то вводит номер телефона
Если номер телефона правильный, то перекинет на ПУНКТ 3

3. Окно с операицями над командам
	1. ЗАРЕГИСТРИРОВАТЬ НОВУЮ КОМАНДУ
	2. НАЙТИ КОМАНДУ (если раннее онную при регистрации не указывал)
	3. ВСТУПИТЬ (если уже в регистрации вводил что-то)
	4. Помощь


3.1 ЗАРЕГИСТРИРОВАТЬ НОВУЮ КОМАНДУ
	= Имя команды
	= Описание команды (описание существующего состава команды и также указание тех скиллов, которые должны обладать новички)


3.2 НАЙТИ КОМАНДУ - выводит список всех доступных команд (т.е. тех, кто в окне управления командой поставили галочку о том, что их МОЖНО найти в общем поиске)
	= [Выводит список команд]

3.2.1 Если человек выбирает какую-либо команду, то командиру той команды отсылается заявка.


3.3 ВСТУПИТЬ - человек вводит id (число из 6 цифр), которое ему скажет капитан. При вводе валидного ID человек сразу же вписывается в команду.
	= ID команды


3.3.1 Если ID верен, то человека сразу же записывает в команду, иначе разворачивает назад.


3.4 ПОМОЩЬ
	- Зарегистрировать команду - создать команду.
	- Найти команду - отобразить список всех доступных команд, и подать заявку на участие
	- Вступить - Если вы уже знаете, в какую конкретную команду вы хотите вступить, то вы можете авторизоваться в ней, минуя процесс поиска и подачи заявки. Для этого вам нужно ввести 6-значный код, который Вам должен сказать капитан интересующей Вас команды.


4. Панель управления командой (для командира)
	1. Регистрация на кейс
	2. Изменить описание
	3. Удалить участника
	4. Удалить команду
	5. Просмотреть заявки
